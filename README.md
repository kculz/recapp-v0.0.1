# RecApp — Rehabilitation & Counseling Client Portal

This workspace contains the initial setup for:
- `/mobileapp`: Client Mobile Application (Expo Router + NativeWind v4) [v1.0.0]
- `/server`: API Backend (Express + Sequelize + PostgreSQL) [v1.0.0]
- `/admin`: Admin & Counselor Web Dashboard (Vite + React + Tailwind CSS) [v1.0.0]

Each directory is a separate node application versioned at `1.0.0`.
